# XmlValidations
Validador XSD de linha de comando para arquivos XML.

O programa recebe em linha de comando dois argumentos: 
 * arquivo.XML. 
 * arquivo de schema.XSD.

# Exemplo de uso:
>> XmlValidations arquivo.XML arquivo.XSD >> resultado.log
